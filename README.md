# computational-statistics-assignments
This repositiory contains python files which are solutions to an assignment in my university's Computational Statistics course.
The questions assigned are listed below:



----------------------------------------
------------- ASSIGNMENT 1 -------------
----------------------------------------

1. Draw a line chart.
2. Draw a semi-logarithmic line chart.
3. Draw a logarithmic line chart.
4. Draw a band chart.
5. Draw a vertical bar chart
6. Draw a horizontal bar chart.
7. Draw a multiple bar chart.
8. Draw a composite bar chart.
9. Draw a pie chart.
10. Draw a pictogram.



----------------------------------------
------------- ASSIGNMENT 2 -------------
----------------------------------------

1. Find frequency, relative frequency, cumulative frequency (less-than type and greater than type) from a set of discrete data.Draw a column diagram for a set of discrete data. 
2. Draw a frequency polygon for a set of discrete data.
3. Draw a cumulative frequency diagram (less-than type)  and greater-than type) for a set of discrete data. 
4. Find frequency, relative frequency, cumulative frequency (less-than type and greater than type) from a set of continuous data. 
5. Draw a histogram for a set of continuous data.
6. Draw a less-than type ogive for a set of continuous data. 
7. Draw a greater-than type ogive for a set of continuous data.




----------------------------------------
------------- ASSIGNMENT 3 -------------
----------------------------------------

1. Given a set of discrete values find the frequencies.  
2. Given a set of continuous values find the frequencies.  
3. Find arithmetic mean, geometric mean and harmonic mean for a set of discrete data. 
4. Find arithmetic mean, geometric mean and harmonic mean for a set of continuous data.



----------------------------------------
------------- ASSIGNMENT 4 -------------
----------------------------------------

1. Consider a set of discrete values. Find the mean deviation about any constant A and about mean. 
2. Consider a set of continuous values. Find the mean deviation about any constant A and about mean. 
3. Consider a set of discrete values. Find the standard deviation and variance. 
4. Consider a set of continuous values. Find the standard deviation and variance. 



----------------------------------------
------------- ASSIGNMENT 5 -------------
----------------------------------------

1. Consider a set of discrete values. Find the first, second, third and fourth raw moments.
2. Consider a set of continuous values. Find the first, second, third and fourth raw moments.
3. Consider a set of discrete values. Find the first, second, third and fourth central moments.
4. Consider a set of continuous values. Find the first, second, third and fourth central moments.
5. Find the measures of skewness.
6. Find the measure of kurtosis.



----------------------------------------
------------- ASSIGNMENT 6 -------------
----------------------------------------

*** MEAN AND VARIANCE ***
1. Read a list of numerical values from a file and compute the arithmetic mean without using any built-in function.
2. Compute the weighted mean of a dataset where both the values and their corresponding weights are are provided in an input file.
3. Calculate the variance of a dataset using the offline (batch) method, and clearly distinguish between population variance and sample variance.
4. Compute the variance of a dataset using the online (streaming) algorithm by processing one data point at a time.
5. Compare the results of offline variance and online variance for the same dataset and comment on any differences observed.




----------------------------------------
------------- ASSIGNMENT 7 -------------
----------------------------------------

*** CORRELATION ***
1. Compute the Pearson correlation coefficient between two variables using the offline method without using build-in correlation functions.
2. Compute the Pearson correlation using an online approach where data points are processed incrementally.
3. Compute the Spearman rank correlation for a dataset by assigning ranks manually, including proper handling of tied ranks.
4. Read a multivariate dataset from a file and generate the complete correlation matrix for all pair of variables.




----------------------------------------
------------- ASSIGNMENT 8 -------------
----------------------------------------

*** LINEAR REGRESSION ***
1. Implement simple linear regression for a dataset with one independent variable and one dependent variable, and display the regression equation. 
2. Compute the Mean Squared Error (MSE), Root Mean Squared (RMS), and coefficient of determination(R^2) for a fitted linear regression model.
3. Implement multiple linear regression with two independent variables and one dependent variables using appropriate mathematical formulations.
4. Implement multiple linear regression with k independent variables using matrix notation and solve it using the normal equation. 
5. Compute the results of your manually implemented regression model with the results obtained using scikit-learn and comment on any differences.




----------------------------------------
------------- ASSIGNMENT 9 -------------
----------------------------------------

*** POLYNOMIAL REGRESSION ***

1. Implement polynomial regression of degree 2 for a dataset with one independent variable and one dependent variable.
2. Extend polynomial regression to higher degrees and plot the resulting curves along with original data points.
3. Determine the saturation power of a polynomial model by fitting models of increasing degree and analysing the change in error. 
4. Demonstrate overfitting by fitting a high-degree polynomial model to a small dataset and compariing with a lower-degree model.
